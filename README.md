# SAT V2.0 - Student Attendance Tracker

## Updates in V2.0
- **Individual Marking:** Clicking 'P' or 'A' now marks only that specific student.
- **Fuzzy Import:** Student name matching improved for Excel uploads.
- **Export Fix:** PDF and Excel exports are now linked to the correct monthly grid data.
- **Visuals:** Dark mode support and refined Navy theme.

## Excel Import Requirements
Your file should have columns named:
- **Roll No**
- **Student Name**
- **Gender**
